怎么将qq好友生日同步到手机日历

发表于 2015-01-06 更新于 2017-09-12 分类于 过去的 Disqus：

今天突发奇想，不，应该说预谋已久，想要把QQ好友的生日一个一个记入到手机的日历上，但是显然工程量庞大啊。遂网上自行搜索了下，发现了一个方法。利用qq邮箱来同步。

1、登陆自己的QQ邮箱

2、查看右侧的生日提醒，点击更多

3、点击最下面的查看所有好友生日

4、第一次进入会提示是否导入所有好友生日，点击是

5、点击手机同步

6、可以看到手机几大系统，选择自己的手机系统，然后按照提示设置

7、一般手机端的日历会自动添加，否则就自己进入日历手动添加邮箱账户。就是这样

发表于 2014-12-31 更新于 2017-09-12 分类于 过去的 Disqus：

那么从何说起呢？因为身在大天朝的原因，而且还听说最近把Gmail的管道都给切断了，想想也是醉了。然后前一段日子买主机空间域名的时候撇到可以利用外国主机搭建代理服务器实现科学上网，这么一想，索性就开始实践了，但是人傻手笨，折腾多番。其实之前并不是没有途径去到外面，但是使用的都是一些免费的vpn类代理软件，比如fqrounter，赛风，易联vpn，电脑端的话就是一些有些fd的代理软件，也有像VPNgate这种样子的。这些优点就是免费，缺点也是免费，因为对于所有人都是如此，那么资源的竞争想必是有的。所以有时候巨慢你！知！不！知！道！但是又买不起那些付费的vpn啊，代理啊，所以每每上网总是郁结。那么为什么要出去呢，额，这个话题嘛比较大，谈不好。那总之就好比有些人为什么要出去旅游一样。当然也因为我自己使用的一些外面的app以及google这样的，所以还是希望能够任性地想出去就出去咯。

如果你看到了这说明你还是对此有所兴趣的，那么我来讲讲具体的一些东西吧，也算是一次记录。

那就从shadowsocks说起吧。

shadowsocks

shadowsocks实质上也是一种socks5代理服务，类似于ssh代理。与vpn的全局代理不同，shadowsocks仅针对浏览器代理，不能代理应用软件，比如youtube、twitter客户端软件。如果把vpn比喻为一把屠龙刀，那么shadowsocks就是一把瑞士军刀，轻巧方便，功能却非常强大。其客户端几乎覆盖所有平台，具体如下，可查看https://shadowsocks.com/client.html 和http://shadowsocks.org/en/download/clients.html

  • shadowsocks-gui - Cross-platform GUI powered by node and Webki

说了这么多，简单一句话，有了shadowsocks账号就可以愉悦地玩耍了！那么怎么获得呢，很简单，要么找要么买。找的话，因为网上会有一些热心奉献的人会共享出自己的账号，以及一些相关的公益组织，这个纯属自己找了。买嘛，那总之有钱总会知道怎么买的嘛，但是也不便宜的说，所以买当中还有一条，就是自己建账号，这就是本文想传达的东西。通过vps自建shadowsocks服务器。

个人总结的个人总结

发表于 2014-12-06 更新于 2017-09-12 分类于 过去的 Disqus：

今天是2014年12月5日，星期五，晴，有风。

不像无数个周末一样，这个周六一大早爬起来去赶赴一场说做就做的广播操比赛。

昨晚却依然像无数个夜晚一样，没有在该入眠的时候入眠。

所以起床是艰辛的，但也必须是坚信的。前几天的衣服搞笑的漫画正给我某种惊醒：帅的人都已经起床，而丑的人还在沉睡。我并不是说我想变得多帅，当然也不是讽刺自己有多丑。它就像一面镜子，别人告诉了我我的样子，而镜子让我知道别人所说的样子，给事物以具化。所以我也要求自己说以后早起不再是事。

但是，我晚上在干嘛。失眠？显然很少有事情能让我失眠。每一个重复的夜晚似乎都是一种迷药，半夜将至，总感觉氛围来了，兴致也来了，还不应该睡觉。睡觉前做点事情似乎是一个伟大的仪式，为了迎接困倦，为了顺其自然的困顿。当然，这点事多半是刷手机，用板刷用牙刷用雨刷。其实，就是简单的刷微薄，刷知呼，刷微信。总像是这一天还剩下点事情没有做完，在等着我去完成，而我也在等着它出现，以此耗费完自己最后的一点精力，才好安然入睡。这样一种状态也不止是我个人而已，相对于整个同龄人抓住几个也多半有这样的情况。因为知乎刷的多的原因，也知道某个大神回答者也是这样的情况，我具体忘了他是如何定义这种现象，应该是某种拖延症，只是他最后战胜了它，让自己的生活进入了一种良性的循环。否则他就不会出现在知乎上了。不过说白了，道理很简单，做起来很难。这也是为什么世间充满道理和知识，成功的人就那么几个的原因。而我现在的想法也寄望于自己的自制力于行动力。

回到早上，边骂着主席台上无关痛痒的老师边做完广播操，吐槽的心谁都会有，只是真正站出来反对的人没有。我当时就在想，看不惯这种没有营养没有意义纯粹是做样子形式主义活动的人不止我一个，但是我不知道是否所有人都是如我所想。因为所有人都没有交流所以所有人都只是心里想想而不会实际行动，因为每个人都怕假如我站出来反对而大多数人选择沉默所以自己也选择成为大多数好了。说来也巧，事后看到一篇微信文章《你被顺从了么》，大概表达的就是这样的事情，只是相对更加鲜明和偏激，语言和内容也更加丰满具体有说服力。

其实这样一个事似乎也是我近来一直想不透的事情之一。

在随后的班会上，要求每个人写一下个人总结，字数不限，对得起良心对得起自己就好。拿着没有笔壳的笔芯，刷刷刷的开始写。前面也了很多矫揉造作的废话，这是我一贯的作风和难以改变的习惯。最后点出几个关键字——“得失参半”。我不好说正好就是五五分，各有一些是必然的。得写的很散很虚，什么交了一些朋友，发现了新爱好摄影，接触了一些以往不曾接触的事情之类的等等。失则写的相对现实具体，奖学金的事成绩的事都没有达到预期，也越来越有一种股市涨停一片信心大失的趋势；虽然有了一些新爱好，但也放弃或者说是不知不觉遗弃了老爱好，乒乓球羽毛球写字等等，一想当年以为的大学：寝室楼下就是体育场，大学四年会参加各种球类，闲暇的时候打打篮球，飙飚乒乓，飘飘羽毛。顿时剩下一声轻蔑的自笑。当然也选择了一些必须要做好选择的东西，思考良久，最终决定的不加共产党。我知道在这个社会，在这个生活中，有些事不是你想做与不想做的问题，更多的是别人和事情推着你向前。我也知道我现在的很多决定未必就是明智的聪明的有效的，但我是个犹豫的人，有时候快刀斩乱麻似乎见效刚好。即使最后有苦头吃，那也只能自己来，怨不得别人也不需要怨。因为这是自己的事，自己的命，要自己把握。而我手上握的筹码还不够重，不如此，也不会有把握更太多筹码的机会。

失去，还有一点没有写上。关于感情，更多的是关于爱情，这个不言而喻。但其实并未算的上是失去，毕竟不曾拥有。但我觉得失去的是自己的青春，失去的是年轻的心，年轻的斗志，年轻的激情。看看现在的自己，似乎更多的是习以为常，逆来顺受了，不是么？所以这样的失去，所占的分量太大。在这条路上走的太难，难得愈发避开了。

现在的状态，当局者迷的意味越发浓郁了。许久前手机的摄像头坏了，觉着是很难修好，怕也不便宜，一直有着某种低沉的郁闷，然后也一直没有去询问售后。拖到今天，当顺利修好后，心情是开朗了，心结也似乎解开了以个小口，看清自己现在的状态，凡事都是如此的拖延，不尝试怎么知道结果，不去做怎么知道接下来的发生，等待着别人的督促永远不会有尽头。看清了现在这样是多么糟糕的一种状态，而槽糕的原因呢，就连自己都没法说清楚。怕是开始解决了才算是看清原因的时候，也是看清原因了才会一点点开始解决。但至少意识到了。这也是我另一个想不透的事情。或者说所有的事情在一起就是一件大的想不透的事情。

现在是2014年12月5日，晚，天黑。

晚饭没吃。似乎更要紧的是被人训一顿。才会懂得已经懂得的道理。

许久过后

发表于 2014-11-30 更新于 2017-09-12 分类于 过去的 Disqus：

突然想起高中时候的MP3，MP4。在晚上走在回来的路上，带着耳机，听着音乐。是感觉还舒服的耳机，Altec lansing的牌子，中文是奥特蓝星，有点土的名字。耳机是最近淘宝刚买的，之前觉得超赞的新浦电声耳机送给妹妹了，配上平板。想起高中那时候听歌的时候怕是正好听着歌，听着那时候就听过的歌吧。耳机里的歌还是那时候的歌，现在的身处，现在的心境怕是一首不会重唱的歌。

记得高中的时候如果要是能有MP3就是很幸福的事情了，当然有些人也有手机。用MP3，载一点喜欢或者流行的歌，不敢再白天拿出来听，多是晚上熄灯了躺床上静静地听一会，坐一会，躺一会，然后睡去。好像有过一个还是两个MP3，小小的那种，应该是两个吧。第一个印象不深，用了一些日子，百来块的样子，耳机是配的，没什么其他要求，能听歌就好。大概能存几十首歌的样子，样子倒是完全忘了。有一个倒是记忆很深，台电的，现在也还在卖，扁长型，两个手指那么大，厚度比手指薄，正面是全玻璃的，一侧嵌着实体按键，背面是银白色。很是小巧，很是精致。用它度过了很多个高中无聊烦闷琐碎的夜晚。后来毕业的时候买了一个也送给了妹妹。

也买过一个MP4，现在或者当时看来，基本上是丑的一逼。好吧，基本上是全山寨还有余，牌子好像是清华紫光来着。显然是用不久的命，也确实如此，拿它看过一些大电影小电影，韩剧动画片什么的，也算是填充了一些漫长的时光。

其实这些东西，当时寝室里基本上人手一个，因为不能带手机，也只有这样聊以慰藉。每个人的牌子质量科技外观等等不尽相同，但基本上都是属于你有那么我也去弄个来的想法。很少有交流的机会，因为设备间不好复制东西，有时候会一撮人挤在一个位置上一起看个电影什么的， 也算是那时候的乐事了。也有事胆子大一点，在床上把外放打开，放放自己喜欢的音乐，唱一唱，笑一笑，或者让大家猜猜歌名。有时候有自己特别喜欢的歌的时候，也会单独凑过来凑过去用耳机来分享分享。当然，也总有被老师抓到现行的，具体多少不知道。总之是有，反正老师办公桌里从来不缺这种东西。不过好险我从未失手过，虽有几次千钧一发。

不过这些事想想也算是自己听音乐的一点小经历。从毫无要求只要能听，到现在耳机至少要是舒服的，当然MP3也早就不用了，怪科学技术发展如此之快，手机已经可以相机播放器等等一起兼并了。只是当年那种在束缚下偷偷摸摸的庆幸与幸福感荡然无存。

近来，或许说有些日子以来，都会带着个耳机，即使没有放着歌。算是一种象征性地隔绝吧。确实耳机能起到的作用是这样的，将你置身于世外，目及的世界与耳听的世界交错融合。仿佛真实，仿佛虚无。所以这也是一部分我总是想有一个舒服的极致的但又性价比高的耳机的原因和目的。不过，耳朵有时候怕承受不了，也是该适时调控。

就像现在，我很享受一个人走在夜晚的路边，校园里，听着耳机里的歌，四周是如此的静谧与安详，听不到寒冷也听不到寂寞，只是跟着音符走动前行移动，在这个世界流浪。只是久了，耳朵会疼想听听世界本来的声音。

许久以后，会怎样的听歌，行走，流浪，思考。

关于第一次毅行那些事（烂尾了，以后再叙旧吧）

发表于 2014-11-03 更新于 2017-09-12 分类于 过去的 Disqus：

走过很多路，依然过不好这一生。

近来好像很忙的样子，总是有很多事情等着去完成，但总是完成不了所有。可是到头来，在某个时刻细想，却发现似乎什么事都为曾做过。

基于这样的原因，断断续续地写着这个博客。这是题外话。

今天是2014年11月3日，昨天是2014年11月2日。

昨天在杭州同时上演着两场好戏，杭州国际马拉松和杭州国际毅行。有幸参加了其中一个，不幸参加了那一个中的艰难的一项。

杭州国际毅行今年是第四届了，往年听说过，但没动身。这次因为一些阴差阳错的原因和几个之前素未谋面的女生组成了一个小组，更是因为因为不知所然的原因选择了毅行50公里。毅行一个分为四个组别：5公里。15公里，30公里，50公里。因为30和50公里属于较长距离，所以组委会规定必须小组报名，四人为一组，小组内必须有一名异性成员。然后我就是那一名异性成员。小组小伙伴都是第一次参加，之所以这么有勇有谋纯属意外，因为报名的时候就剩下50公里名额了，然后当时是早上大家都没睡醒，一定是这样。

因为事先没有买组委会的交通车票，所以我们是自己乘车到达起始会场。前一晚，也就是1日晚上，或许可能太兴奋也或许平时这个点本来就没睡，一直玩耍到12点多才躺下，然后还辗转反侧。然而闹钟是5点的，是的，你没看错，所以估摸着也就睡了4个小时多的样子，然后闹钟一响利索地爬起来，冲向已有小伙伴在等待的公交车站。天下着小雨，也还没有太多光亮，然后在一刹那道路两旁的路灯突然熄灭，意味着新的一天确实是要开始了。

八点开始，七点到达。找到组织，静待各种流程流过，然后，走。

天公还算作美，多云转晴。当会场的主持人一声令下，所有人朝着会场后方的大门入口涌出去，当时在想真像是一群劳改释放的人激动地要出去看看外面的蓝天白云啊。一开始纯粹就是被挤着出去的。组委会用衣服来区别不同的组别，这样方便识别和引导，因为大家起点相同但路线和终点不尽相同。黄色是5公里组的，很多都市的年轻家庭带着小孩穿着亮绿色的衣服。15公里组是亮绿色，很多情侣和中年夫妇。30公里是橙红色，50公里是绿色，这两个组别大多都是年轻学生和中老年人，据我观察，年轻学生都是一些不知所然年少无知的那种满打满撞的，反正年轻就是力量，年轻就是行的德行；而中老年人显然更成熟稳重，平时也运动锻炼有佳，一路上谈笑风生，轻松加愉快。所以说尽快组别是一样的，但是心态与心境却是截然不同的。

比赛似乎来得太突然的感觉，当所有人像一条河流在钱塘江边涌动的时候，还没做好准备的膀胱开始有了尿意，然后厕所也似乎没做好准备，才2公里开外的第一个厕所，男女厕所门口俨然长龙，不失为一种观景。索性忍一忍继续前行。比赛中的打卡，应该不用多说，无非就是走到某一个点敲章签到，证明我到了，然后集合所有的敲章就可以召唤龙神，哦，不，是召唤奖牌和荣誉证书。但是打开点的设置却多少让人凌乱。设置并未按照整数来设置，反正走到了5公里，第一个打卡点还在前方。而打开点、输水点、医疗点、收容点也像是一种奇葩组合，虽然错开了，但总让人看到希望看到失望。因为往往看到了一个摆摊帐篷就以为前方是打卡的地方，而有时却是输水或者其他。

说实话，在刚开始的5公里，小腿显然很快就出现了异样，似乎一不小心就在下一秒就会抽筋硬化，因为时不时地会抽搐一下，这就好像有时候有呕吐感，但是其实有时是干呕，而你也确实不知道下一次会不会吐出点什么来。但是当你看到一路上有小孩子在蹦蹦哒哒，有老年人在健步如飞，那羞愧的心强迫着你必须坚持前行，同时安抚那躁动的抽搐。

走过第一个打卡点大概花了一个多小时，然后集体向前寻找厕所，对，又是厕所。因为没有发现公共厕所，所以尾随其他人进入了边上建在钱塘江上的海事办事厅上厕所。可是，只能说不能更醉了，里面的厕所是男女共用的那种，所以你就看到又是一条长龙，而且男女夹杂。所以索性就当中途休息，开始在那排队，估摸着排队排了半小时多，然后不禁愤愤地吐槽组委会的组织是有多么有力啊！

一身轻松后继续上路，一路上大家为了缓解气氛缓解酸痛，就开始能聊什么聊什么，看见什么聊什么，说无聊其实看看风景看看人群也是蛮有意思的，说有聊那小腿上的紧张感分分钟想让你停下来休息。走过大概10公里的时候显然小腿更加造反了，疼痛开始又小腿向上转移到膝盖下边一点，并伴随着大腿与屁股之间。队友三个，其中两个似乎乐在其中，虽然也看到他们偶尔很别扭地走出几步想更舒服的姿势步伐，但时不时地像两只小鹿，就欢快地向前跑去然后继续走。而我则和另一个稍微腿短的队友（好吧，其实我也腿短，腿长真心有优势）默默地走在后面，也时不时地小跑一下。根据我自己的分析，当人长时间行走，下身的运动方式完全是属于机械式地重复，势必会造成某几个地方疲惫酸痛，如果改变一下运动方式会稍微缓解这种情况，所以当有些酸楚疼痛的时候小跑一下或者改变脚与地面的接触方式等就能舒服一点了。然后事实似乎也确实如此，因为每当我小跑一阵再继续走，就感觉好多了，想想还是好机智。

到达第二个打卡点已是中午。我们领了免费的红牛，坐下来打算吃午饭。东西是昨日特意去买的，都是一些高能量含糖含热量多的事物，显然有些不好吃。但是简单的补充了下就又感觉瞬间能力满满了呢，显然那一罐红牛，就有点像玩全民飞机吃了个爱心一般加血了，但是一站起来走一步，呵呵，痛，都不知道哪里痛了，然后下午就这样开始了。
