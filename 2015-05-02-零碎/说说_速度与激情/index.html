说说 速度与激情

发表于 2015-05-02 更新于 2017-09-30 分类于 零碎 Disqus：

/有木有发现博客外观变了一点，额，基本看不出来，反正我修改了下/
说说速7，赶在了大概一个月后快下架的时候，独自跑到电影院看的。
以前零零散散地看过了速度与激情系列的前六部，不知道是什么时候看的，情节也忘得差不多了。所以当速7快要上映的时候，我说，重新把这个系列再看一遍吧。那是在保罗沃克去世了之后。不过这有个插曲是，今天发现速度与激情系列电影第一部导演是美国导演罗伯·科恩，第二部是美国导演约翰·辛格顿，第三部到第六步居然一直是华裔导演林诣彬，而第七部又是华裔导演温子仁。
于是在四月的某一天直接用超快网速下好了前六部，准备空下来一部一部慢慢观赏。所以四月有六天是很兴奋的，受电影感染。不过对于这个系列，观看的顺序却不是按上映时间来的，也就是说不是按照123456来看的，准确的说应该是1245637。据我猜测，第三部这个奇葩的出现可能是当时电影公司或是演员出了什么问题吧。那么让我来做个我理解的这个系列的总结吧。

  1. 第一部
    一伙人开始认识。Toretto和妹妹Mia在小镇开着小店，Toretto还带着一帮兄弟。这时Brian作为警察卧底过来调查事情。然后他们一起参加飙车大赛，玩着玩着Brian就发现自己过来的正事，Toretto他们抢劫大货车赚外快，但是Brian没有抓他们反而救了他们还放走了Toretto，当然爱上他妹妹Mia是关键。
  2. 第二部
    Brian直接不做警察了，因为第一部的失职。但是呢，警察又有需要找他帮忙抓一个洗钱的家伙。于是他就找了他小时候可爱的小伙伴Pearce当送钱的司机帮坏蛋送钱然后人赃俱获。额，好像就这样，其中的卧底的妹纸好像哪里又出现过的样子。
  3. 第四部
    Brian又做回了警察啊我去。因为第一部里面Toretto犯事了所以在外国避难。他的妹纸Letty为了让他回来和警察达成协议卧底抓坏蛋，结果死了。Toretto很生气回来了，阴差阳错又和Brian碰头。为了抓那个坏蛋，贩毒的，这回两人是当送毒的司机帮坏蛋送毒品，跨境的。最后抓了坏蛋，Toretto也进去了，认识了开车的新妹纸Gisele。但是！Brian劫囚车啊卧槽又不当警察了啊，然后就要跳到下一部去了。
  4. 第五部
    现在一伙人都是犯事的人了吧，就逃到了巴西里约。没想到美国大汉Hobbs追到了里约。碰巧他们因为干了一票买卖惹了当地的大亨，Toretto找来好些人，其中有Han，还有个很厉害的电工，两伙人打了起来，不，是三伙人打起来，最后是两伙人一直对抗大亨，直接偷走了他藏钱的保险箱，一间小房子那么大。最后大家拿着好多钱变成了千万富翁，从此过上了幸福快乐的日子。耶~
  5. 第六部
    混蛋，Letty没死啊，就是Toretto女票或者说老婆。原本幸福生活的一群人又出来了。这次对付的是一伙正规军一样的国际罪犯，Letty因为没死是失忆了所以也在里面。一番恶斗，美国大汉Hobbs和Toretto一群小伙伴（大概就是之前的一伙）打败了他，救回了Letty。
  6. 第三部
    终于是这个坑爹的一部了，其实不坑呵呵。Han拿了那么多钱后去了东京，因为第六部的时候死了Gisele，他爱的妹纸，所以人生暗淡。在东京教一个美国来的小屁孩漂移，但是因为做生意坑人追杀中被撞死，但是撞人的却是第七部的大BOSS，然后小屁孩为了解决报仇又开始比赛车最后自然赢了。最后出现Toretto，预示第七部的事情。
  7. 第七部
    终于这个了，好累，好不会总结。继第三部那个撞人的人，就是这部大BOSS，是第六部大BOSS国际罪犯的大哥哥，回来报仇的说。两伙人又是腥风血雨啊。但是冒出个xx政府组织说要找一个很厉害的黑客和她做的芯片，能监听和查找世界上任何人。帮助他们就能抓住大BOSS，于是他们边救人边对抗大BOSS，好吧这个我表示蛋疼。最后各种车技开车又赢了。最后的最后大家终于过上了幸福快乐的日子。

额，我的总结好苍白无力啊（哭。
明天用思维导图做个关系图吧。
其实对于第七部是有很多期待的，但是对于这样的好莱坞商业片，你说看出多少人生意义却又是不实在的。吸引的是那种拼搏的闯劲和团队一起的干劲儿。很经典的是Toretto对于家庭的定义，也是对这个团队的定义。大致意思是：人生没有太多的东西能够拥有，但是最大的财富就是现在与我一起举杯的屋檐下的你们。其实，纯粹电影来讲，这么多部电影，所有行为的目的有些很牵强，但是过程很精彩，内容很丰富，精神很可嘉。所以观众也很开心欢快激情激动。毕竟故事就是故事，电影就是电影，不同于真实。
或许一部电影能带给一个人很多思考，何况一系列。于我，现在，生活需要刺激也需要激情，需要有闯荡的勇气也需要温暖的背靠。所以为之努力。
