github小记

发表于 2015-02-26 更新于 2020-03-17 分类于 零碎 Disqus：

拖了许久的web开发总算是好死不活的把它做好了，虽然还有一些bug，但是也不管了，等以后升级更新吧。然后今天还做的一事就是把这货同步到github。github大家都知道，托管代码的好地方。之前我把一些代码啊什么的备份到金山快盘，没成想晚上打开，发现这货居然外链无法启用（国情审查说），也是无奈。想找个好一点的网盘备份东西，国内的不靠谱，国外的被封杀。不说也罢。于是乎又逛了一圈的网盘，顺便给这个FarBox找个图床噻。毕竟有数量限制。
还是说回github先，这坑爹的，下载好github for windows，默默安装好了，TMD居然无法登录，折腾半天，一开始以为是代理问题，于是挂了代理，后来发现不是，网上一查说是.net版本问题，发现已经是最新版。后来的后来，突然鬼使神差地发现可能是password这个隐藏输入有问题，输入框内的密码未必是我所想输入的密码，于是在他处输入好密码再复制过来，果不其然啊卧槽，就这样解决了。附图。

其实似乎我在解决github这个问题上花的时间也就那么一点，反而在给farbox找图床这事上花了好久好久，就在这图能显示之际才算踏实地找好。最后的决定是使用：七牛云。附带上它的上传小软件，也不算是一种方法。虽然farbox直接拖拽的插入方法也是逆天。但是我就是这么喜欢不找边际的倒腾！

不早了，不迟了。附张美图睡觉。
